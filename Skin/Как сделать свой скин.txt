   В каталоге программы есть подкаталог "Skin". По-умолчанию в нем нет 
графических файлов, и Orix использует собственные фоновые рисунки в 
окнах программы. Если Вы хотите изменить фоновый рисунок в каком-либо 
окне программы, то необходимо поместить в подкаталог "Skin" графический 
файл типа "jpg" с соответствующим именем. Имя для данного файла определяется 
следующим образом:
    
   1)Для изменения фонового рисунка окна "Сегодня и скоро" - pic1.jpg
   2) Для окна "Уже было" - pic2.jpg
   3) Для окна "Архив дел" - pic3.jpg
   4) Для окна "Редактор данных" - pic4.jpg
   5) Для окна "Установки" - pic5.jpg
   6) Для окна "Перспектива" - pic6.jpg
   7) Для окна "Сводный отчет" - pic7.jpg
   8) Для окна "Записи" - pic8.jpg
   9) Для окна "Редактирование записи" - pic9.jpg

   Размер фоновой картинки должен быть от 950 x 500 и более. Программа отобразит картинку 
без масштабирования начиная с ввернего левого угла.
